# FaceMatch / 妆仿 — 产品规划书

## 一、产品定位

**一句话：** 拍别人的妆 → AI 迁移到你脸上 → 看效果 + 学教程

**Slogan：** 看见你的妆，复制你的美
**Slogan (EN)：** See the look. Make it yours.

**差异化 vs GlowUp：**
- GlowUp 偏「选模板试妆」，FaceMatch 主打「任意照片都能仿」
- GlowUp 强在教程+产品推荐，FaceMatch 强在妆容迁移精度
- 支持 Android + iOS（GlowUp 目前 iOS only）

| 人群 | 场景 | 优先级 |
|------|------|--------|
| 18-30 岁女性 | 刷到明星/博主妆容想试试 | P0 |

## 三、功能分期

### 🟢 P0 MVP（6-8周）

**核心闭环：选参考 → 上传自拍 → 生成 → 看效果**

```
1. 拍照/选图（参考妆容）
2. 拍照/选图（自己的脸）
3. AI 识别妆容要素
4. AI 迁移到目标脸
5. 展示效果图（前后对比）
6. 保存/分享结果
```

**具体功能：**
- [x] 相册选图 + 拍照（参考图 + 自拍）
- [x] AI 妆容分析（识别眼影/口红/腮红/眉形/底妆等维度）
- [x] AI 妆容迁移生成（出 1-3 张效果图）
- [x] 生成结果展示 + 放大查看
- [x] 保存到相册 / 分享
- [x] 免费额度限制（每天 3 次免费生成）

### 🔵 P1 增强（10-12周）

- [ ] 妆容参数手动微调（调口红颜色/眼影深浅）
- [ ] 分步化妆教程（AI 根据你的五官生成步骤）
- [ ] 推荐匹配产品（根据肤色/妆效推荐化妆品）
- [ ] 生成历史记录
- [ ] 支持从视频截图仿妆

### 🟣 P2 社区化（16周+）

- [ ] 用户上传「今日妆容」供别人仿妆
- [ ] 妆容排行榜 / trending
- [ ] 收藏妆容配方
- [ ] 关注美妆博主
- [ ] AI 妆容相似度评分（你这个妆仿得像不像）

### ⚪ P3 进阶

- [ ] 实时摄像头试妆（AR）
- [ ] AI 发型迁移
- [ ] AI 美甲设计
- [ ] 品牌合作专区（植村秀/雅诗兰黛等官方试色）

## 四、技术架构

```
┌─────────────────────────────────────┐
│           前端 (Flutter)              │
│  - iOS + Android 双端                │
│  - 相机/相册权限                      │
│  - 图片裁剪/预处理                     │
│  - 结果展示/分享/保存                  │
└──────────┬──────────────────────────┘
           │ REST API
┌──────────▼──────────────────────────┐
│          后端 (FastAPI)               │
│  - 用户管理 / 订阅管理                │
│  - 生成任务队列                       │
│  - 妆容数据 CRUD                      │
│  - 社区功能                           │
└──────────┬──────────────────────────┘
           │
┌──────────▼──────────────────────────┐
│         AI 推理层                     │
│                                      │
│  Step 1: 妆容解析                     │
│  模型: GPT-Image-Vision / Gemini Vision│
│  输入: 参考妆容图                     │
│  输出: 结构化妆容参数                  │
│  { eye_shadow: "暖棕系, 大面积晕染",   │
│    lipstick: "砖红色, 哑光",           │
│    blush: "杏色, 颧骨外侧", ... }     │
│                                      │
│  Step 2: 妆容迁移生成                  │
│  方案A: GPT-Image-2 (开箱即用, 按量付费)│
│  方案B: Stable Diffusion + ControlNet │
│         (自部署, 成本更低)             │
│  方案C: MiniMax image-01 (备用)       │
│                                      │
│  输入: 素颜自拍 + 妆容参数             │
│  输出: 仿妆效果图                     │
└──────────────────────────────────────┘
```

### 推荐方案（起步）

```
用户上传参考图 → Gemini Vision 分析妆容 →
GPT-Image-2 以自拍为底图 + 妆容描述生成效果
```

**理由：** 不需要自训练模型，两个 API 调用就能跑通全流程，MVP 最快上线。

**成本估算（单次生成）：**
- Gemini Vision 分析 ~ $0.01
- GPT-Image-2 生成 ~ $0.05-0.10
- ≈ $0.10/次，按免费 3 次/天，一个月成本 ~ $9/用户？不对，得设置合理免费额度

实际上：
- 每天免费 3 次 → 轻度用户够了
- 重度付费用户无限畅用
- 每个付费用户月均 30-60 次生成 → 成本 $3-6
- 订阅 $9.99/月 → 毛利 40-70%

## 五、变现方案

### 定价模型

| 方案 | 价格 | 权益 |
|------|------|------|
| 免费 | $0 | 每天 3 次仿妆 + 基础效果 |
| 每周 | $3.99 | 无限生成 + 微调参数 |
| 每月 | $9.99 | 无限生成 + 微调 + 教程 + 产品推荐 |
| 每年 | $49.99 | 所有功能 + 未来新功能抢先体验 |

### 其他收入

- **单次购买**：$0.99/次额外生成（不想订阅的偶尔用户）
- **品牌合作**：产品推荐里植入品牌（植村秀/完美日记等），CPC 或 CPA
- **妆容滤镜包**：付费解锁特殊风格妆容包（日系/欧美/复古等）
- **Members Club 邮件**：推送优惠/新品（参考 GlowUp）

## 八、关键风险

| 风险 | 概率 | 影响 | 应对 |
|------|------|------|------|
| 妆容迁移效果不够好 | 中 | 高 | MVP 先用 GPT-Image-2，后续切自部署 SD |
| API 成本偏高 | 中 | 中 | 按量定价 + 免费额度控制，长尾用自部署模型 |
| 隐私敏感（用户上传素颜照） | 高 | 高 | 端侧处理承诺、不上传存储、可匿名使用 |
| 苹果审核（妆容/医疗健康类） | 低 | 中 | 明确不做医疗诊断，预先准备免责声明 |
| GlowUp 先发优势 | 高 | 中 | 差异化定位（任意图仿妆），不做跟风 |
